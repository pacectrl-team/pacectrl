# PaceCtrl
* [backend](backend) folder will be used for FastAPI. Will be split into public (the one used by the widget) and not so public, used by operator backend and the portal
* [portal](portal) folder will be used for the portal used by operators to create new trips, manage settings (like widget thresholds), invite crew. Has a login screen.
* [widget](widget) folder will be used for the embeddable script source + build output. This is where the widget lives. Default design. Operator can then edit in the portal the settings (thresholds) and their own branding to it, like colors and font.
