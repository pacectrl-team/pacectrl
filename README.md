# PaceCtrl
* [backend](backend) folder hosts the FastAPI service (public widget APIs, operator APIs, and static widget bundle delivery).
* [portal](portal) folder will be used for the portal used by operators to create new trips, manage settings (like widget thresholds), invite crew. Has a login screen.
* [widget](widget) folder contains the embeddable script source + build output. This is where the widget lives. Default design. Operator can then edit in the portal the settings (thresholds) and their own branding to it, like colors and font.
